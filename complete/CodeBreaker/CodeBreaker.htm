CodeBreaker

Instructions

The aim of the game is to guess the hidden code. The hidden code is between 4 and 8 symbols in length, there are between 6 and 10 symbols in the game, and you have between 8 and 12 attempts to guess the correct code. The code length, number of symbols and maximum number of guesses are set during game setup.

To enter an input code, left- or right-click on each symbol in the input code area to cycle forward or backward through the available symbols. Once you are satisfied with your guess hit the 'Submit' button.

The input code you submitted will be shown at the top of the results list, and will be colour-coded to indicate the accuracy of the guess, as follows:

 The input symbol matches the symbol at the same position in the hidden code
The input symbol occurs in the hidden code but at a different position
The input symbol does not occur in the hidden code
