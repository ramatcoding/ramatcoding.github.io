Press CTRL + SHIFT + J (On Windows and Liunx), CMD + SHIFT + J (Mac)!

Type GDFTOTAL in the console.

Then type GDF in the console.

After you type GDF in the console you'll see the function of the GDF.

Now type getDistance in the console.

Now you see the getDistance function.

After that type speed.

Now you see the speed function.

Finally type time in the console.

You see the time function.
